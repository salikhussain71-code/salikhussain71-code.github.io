## About

This site is the canonical home for my academic and research identity online. It tracks my work in real time — projects, certifications, technical skills, and a running journal — rather than presenting a static, one-time snapshot. It exists to give professors, admissions committees, and collaborators one accurate place to see what I have actually built, with links back to verifiable sources (GitHub repos, credential IDs, live demos).

**Flagship research project:** [PAKGOV-RAG](https://github.com/salikhussain71-code/pakgov-rag) — a bilingual Urdu-English retrieval-augmented generation system for Pakistani government documents (FBR, HEC, NADRA, SECP, Supreme Court).

## Tech Stack

Built deliberately simple — no framework, no build step, so it stays fast, dependency-free, and easy to maintain for the next four years:

- **HTML5 / CSS3** — single static file, custom design system (no CSS framework)
- **Vanilla JavaScript** — scroll-based section highlighting only
- **Fonts:** Fraunces (display), Inter (body), JetBrains Mono (metadata/code)
- **Hosting:** GitHub Pages (free, static)
- **SEO:** `robots.txt`, `sitemap.xml`, Google Search Console verification
